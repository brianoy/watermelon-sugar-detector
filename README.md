# watermelon-sugar-detector

我也不知道我在寫什麼，大學時期的一個小專案，發想是買西瓜的時候長輩都會拍打西瓜，因為糖分影響密度，使得有些西瓜聲音比較低沉(咚咚咚)，有些西瓜聲音比較高(啪啪啪)。以此概念，藉由ftt再透過機器學習的決策樹，最後因為西瓜太貴了，我們只買得起5顆，樣本數嚴重不足(5顆)，即便拍打原西瓜，每次輸出結果還是不一樣，實驗以失敗告終。
